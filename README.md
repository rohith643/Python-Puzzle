**Sliding Puzzle Game (3x3)**
A classic 3x3 sliding puzzle game built using Python and Tkinter. Shuffle the tiles and try to arrange them in order by sliding them into the empty space. A fun way to test your logic and problem-solving skills!
**Features**
- Interactive GUI using Tkinter
- Randomized, always-solvable puzzle generation
- Move counter to track your progress
- "New Game" button to reshuffle and restart
- Win detection with congratulatory popup
**Requirements**
- Python 3.6 or higher
- Tkinter (comes pre-installed with most Python distributions)
  **How to Run**
 **Clone the repository:
git clone https://github.com/rohith643/Python-Puzzle/edit/main/README.md
cd sliding-puzzle-game**
**- Run the game:
python sliding_puzzle.py**
